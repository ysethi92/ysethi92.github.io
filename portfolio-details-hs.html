Project information

  • Name: Hate Speech Detection
  • Category: Natural Language Processing (NLP)
  • Project date: December 30, 2021
  • Project URL: Github

  • Implementated multiple ML and DL based model for detecting hate speech in english text.
  • The models are implemented using tensorflow and Pytorch.
  • SVM with Universal Encoder Sentence Embeddings gave the best results.
Dataset Source: SemEval 2019 - Task 5
